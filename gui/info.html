Vault Usage Guide

Move In Files & Create New Directories

  • To move in files, you could right-click on the blank space, choose "Move In File", and upload files through the "File Explorer" , or you could simply drag and drop the file into the "dropzone" , which will directly upload the file to the corresponding directory.
  • To create a new directory, simply right-click on the blank space and choose "Create New Directory".

Edit Files & Directories

  • Right-click on a directory and choose "Rename" to rename the corresponding folder.
  • Rename operation on a single file is NOT permitted due to security rules.

Move Out Files & Remove Directories

  • Similar to the move-in operation, right-click on a single file and choose "Move Out File" to remove file through the "File Explorer" .
  • Right-click on a directory or a file and choose "Delete" to delete the corresponding folder or file. Warning! This operation is IRREVERSIBLE!
